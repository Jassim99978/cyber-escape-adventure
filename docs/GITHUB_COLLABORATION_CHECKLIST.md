# GitHub Collaboration Checklist

Use this in the real GitHub repository before final submission.

## Repository Rules

- [ ] No direct pushes to `main`
- [ ] Every code change goes through a pull request
- [ ] Final version is tagged as `v1.0` or clearly labeled as the final commit

## Team Requirements

- [ ] GitHub repository link is ready
- [ ] At least 8 GitHub Issues were created
- [ ] Issues were assigned to team members
- [ ] `docs/flowchart.png` is uploaded to the repo
- [ ] README is complete

## Per-Student Requirements

Each student needs:

- [ ] At least 2 pull requests
- [ ] At least 6 real commits
- [ ] At least 2 PR reviews
- [ ] Code contribution, not only story text
- [ ] Individual survey completed
